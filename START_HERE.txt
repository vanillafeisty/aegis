================================================================================
                            🎯 QUICK START (5 MIN)
================================================================================

1. Extract the ZIP:
   unzip aegis.zip && cd aegis

2. Create environment file:
   cp .env.example .env

3. Add your Gemini API key to .env:
   GEMINI_API_KEY=your_new_key_here

4. Start everything:
   docker-compose up -d

5. Open browser:
   Frontend:    http://localhost:3000
   API Docs:    http://localhost:8000/api/docs
   Grafana:     http://localhost:3001 (admin/admin)

6. Register & explore:
   Sign up at the login page and start using Aegis!

================================================================================
                        ⚠️  SECURITY REMINDERS
================================================================================

BEFORE RUNNING:

1. ✅ Rotate your exposed credentials:
   - Delete old Gemini API key → Create new one
   - Change LinkedIn password
   - Check AWS for unauthorized activity

2. ✅ Use NEW credentials in .env (not the exposed ones)

3. ✅ Verify .env is in .gitignore

4. ✅ NEVER commit .env to git

5. ✅ Keep SSH keys secure
